⚖️ hanlaw-skill 빌더

한국 법령·판례를 Claude에서 바로 조회하는 Skill

원본: SeoNaRu/lexguard-mcp (MIT) · 포팅: dogdogfat/hanlaw-skill

이 페이지는 뭔가요?

Claude Pro 사용자가 Cowork 환경에서 한국 법령·판례를 조회할 때 API 한도 걱정 없이 쓸 수 있도록 만든 Claude Skill 빌더입니다. 본인의 OpenLaw API 키를 입력하면 그 키가 포함된 Skill ZIP을 이 브라우저 안에서 바로 만들어 다운로드할 수 있어요.

⚠️ 면책: 법률 자문을 대체하지 않습니다. API 결과는 참고용이며, 최종 판단은 변호사·법무사 등 전문가와 상의하세요.
1

OpenLaw API 키 발급받기

국가법령정보센터(open.law.go.kr)에서 무료로 발급받을 수 있고 신청 즉시 자동 승인됩니다.

  1. open.law.go.kr 접속 → 우측 상단 [사용자 가입] 클릭, 개인정보 입력 후 가입
  2. [로그인] 후 상단 메뉴 [OPEN API] 클릭
  3. 좌측 메뉴 [OPEN API 신청] 진입 → [새 OPEN API 신청하기] 버튼 클릭
  4. 신청서 작성 — 기업명(개인이면 "개인"), 신청자, 시스템/홈페이지명, 도메인주소(개인이면 "도메인 없음"), 그리고 활용목적(필수) 입력 후 제출
  5. 자동 승인 처리 — 바로 사용 가능 즉시
  6. API 키 확인:
    마이페이지 → API인증값변경, 또는 OPEN API → OPEN API 신청 목록에서 확인·복사

⚠️ 주의: 활용목적을 공란으로 두면 승인이 취소될 수 있습니다. "개인 학습용 Claude Skill 연동" 등 구체적으로 기재하세요.
📞 기술 문의는 open.law.go.kr 하단 "이용문의" 연락처 참조.

2

내 API 키로 Skill 만들기

사용 중인 OS를 선택하고 발급받은 API 키를 붙여넣으세요.

운영체제 선택

🔒 개인정보 보호: 입력한 API 키는 이 브라우저 안에서만 처리되며, 외부 서버로 전송되지 않습니다. ZIP 생성은 JavaScript(JSZip)로 클라이언트 사이드에서만 이뤄집니다.
3

Claude에 Skill 설치하기

✨ 방법 1 — Claude 앱에서 ZIP 업로드 추천

OS와 무관하게 가장 빠른 방법. 다운로드한 ZIP을 그대로 불러오면 끝.

  1. Claude 앱 사이드바에서 [Customize](커스터마이즈) 클릭
  2. [Skills] 탭 선택
  3. 우측 상단 [+ 추가] 버튼 클릭 → [스킬 업로드] 선택
  4. 방금 다운로드한 hanlaw-skill-mac.zip / hanlaw-skill-win.zip 을 업로드
  5. Skill이 자동 등록됨 → 새 대화에서 테스트:
    근로기준법 제50조 내용 알려줘

💡 이 방법은 재시작이 필요하지 않고, 파일 탐색기로 경로를 찾을 필요도 없습니다.

🛠 방법 2 — 파일 탐색기/Finder에서 수동 복사 (고급)

🍎 macOS / Linux

  1. 다운로드한 hanlaw-skill-mac.zip을 더블클릭해 압축 풀기
  2. Finder에서 Cmd+Shift+G → 아래 경로를 붙여넣기 (없으면 직접 생성): 
    ~/.claude/skills/
  3. 압축 푼 hanlaw-skill 폴더 전체를 위 skills/ 안에 복사
    → 최종 경로: ~/.claude/skills/hanlaw-skill/SKILL.md
  4. Claude Desktop/Code 재시작

🪟 Windows

  1. 다운로드한 hanlaw-skill-win.zip을 우클릭 → [압축 풀기]
  2. 파일 탐색기 주소창에 아래를 붙여넣고 Enter (경로 없으면 직접 생성): 
    %USERPROFILE%\.claude\skills
  3. 압축 푼 hanlaw-skill 폴더 전체를 위 skills 폴더 안에 복사
    → 최종 경로: %USERPROFILE%\.claude\skills\hanlaw-skill\SKILL.md
  4. Claude Desktop/Code 재시작

📡 공인 IP 등록 (응답이 없을 때)

OpenLaw API는 호출 PC의 공인 IP가 등록되어 있어야 응답합니다. Skill 설치 후 "응답 없음" / "IP 미등록" 오류가 나오면 아래를 진행하세요.

  1. 공인 IP 확인:
    🍎 macOS/Linux (터미널): curl -s https://api.ipify.org
    🪟 Windows (PowerShell): 
    (Invoke-WebRequest -Uri https://api.ipify.org).Content
  2. open.law.go.kr → 마이페이지 → [API인증값변경] → 위에서 확인한 IP 추가

자주 묻는 질문

Q. 내 API 키가 GitHub나 어딘가에 저장되나요?

아니요. 이 페이지의 JavaScript가 키를 받아 브라우저 메모리에서만 ZIP 파일을 조립합니다. 네트워크 요청이 전혀 발생하지 않습니다 (템플릿 파일 로딩 후에는 오프라인에서도 동작).

Q. Claude Pro가 아닌데도 쓸 수 있나요?

네. Claude Free/Pro/Max 모두 Skill 기능이 있으면 사용 가능합니다. 다만 이 프로젝트의 주 동기는 Pro 사용자가 Cowork에서 MCP 서버 비용 없이 한도 걱정 덜하게 쓰는 것입니다.

Q. 소스 코드를 직접 수정하고 싶어요

GitHub 저장소를 clone해서 skill/scripts/lexguard_api.py의 __LAW_API_KEY__ 부분을 직접 수정하면 됩니다.
